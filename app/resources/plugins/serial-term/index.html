Connection settings

📖 Manual & help

Terminal color coding

  • White — text being edited (sent on Enter)
  • Yellow — text sent to the serial port
  • Green — text received from the serial port

Options

  • Local echo: also print the data sent to the terminal.
  • Flush on Enter (checked): send the data only when Enter is pressed. Unchecked: send every character as it is typed.
  • Hex view (checked): show received data as hexadecimal instead of ASCII.
  • On Enter sends: special character(s) appended to the TX stream on each Enter (CRLF / CR / LF / None).

Keyboard shortcuts (editing)

  • ← →: move the cursor left/right.
  • ↑ ↓: move up/down (multi-line continuation mode).
  • Delete Backspace: delete one character.
  • Home End: jump to start/end of the input.
  • Alt+↑ / Alt+↓: command history (like bash). At the start/end of the text, ↑/↓ also browse history.
  • Alt+← / Alt+→: move by word.
  • Alt+Backspace: delete a whole word.

Multi-line continuation

A command can span several lines when it contains unbalanced quotes, a boolean operator (&& or ||), a pipe (|), or a trailing line-continuation (\).

Select, copy & paste

  • Select: drag with the mouse, double-click (word) or triple-click (block).
  • Copy: Ctrl+Insert or right-click → Copy.
  • Paste: Shift+Insert or right-click → Paste.

Remark: do not copy text from the white (editing) section — carriage returns there are replaced by spaces.
